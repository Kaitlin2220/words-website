# 英语单词复习网站 - 交互设计

## 核心交互功能

### 1. 智能单词学习系统
- **单词卡片模式**: 左右滑动浏览单词，点击翻转查看释义
- **发音功能**: 点击音标或按钮播放标准英音/美音发音
- **难度分级**: 用户可标记单词难度（简单/中等/困难），系统根据记忆曲线推荐复习
- **学习进度**: 实时显示当前单元学习进度和整体完成度

### 2. 多样化练习测试
- **英译中测试**: 显示英文单词，选择正确中文释义（4选1）
- **中译英测试**: 显示中文释义，拼写英文单词
- **听力测试**: 播放单词发音，选择正确拼写或释义
- **完形填空**: 在句子中填入正确的单词形式
- **限时挑战**: 60秒内答对尽可能多的题目

### 3. 游戏化学习
- **单词消消乐**: 拖拽相同释义的单词配对消除
- **单词拼图**: 将打乱的字母重新排列成正确单词
- **记忆翻牌**: 翻转卡片匹配单词和释义
- **单词接龙**: 根据最后一个字母连接下一个单词

### 4. 学习数据统计
- **个人词库**: 显示已学习、掌握、待复习的单词数量
- **学习日历**: 记录每日学习时长和单词数量
- **错误本**: 自动收集做错的单词，重点复习
- **成就系统**: 学习里程碑徽章和连续学习奖励

### 5. 智能复习推荐
- **遗忘曲线**: 基于艾宾浩斯遗忘曲线安排复习时间
- **个性化推荐**: 根据错误率推荐重点复习单词
- **单元测试**: 每个单元学习完成后自动生成测试

## 用户交互流程

### 主页面学习流程
1. 选择学习单元（Unit 1-8）
2. 进入单词卡片浏览模式
3. 点击发音、标记难度、翻转查看释义
4. 完成单元学习后进入练习测试
5. 查看测试结果和错误分析

### 练习模式流程
1. 选择测试类型（英译中/中译英/听力/完形填空）
2. 设置题目数量（10/20/30题）
3. 开始计时测试
4. 实时显示答题情况
5. 提交后显示详细解析和正确答案

### 游戏模式流程
1. 选择游戏类型（消消乐/拼图/翻牌/接龙）
2. 选择难度级别和单元范围
3. 开始游戏，计时计分
4. 游戏结束后显示得分和排名
5. 可重复挑战或选择其他游戏

## 数据持久化
- 使用localStorage保存用户学习进度
- 记录每个单词的学习状态（未学习/已学习/已掌握）
- 保存测试历史和成绩数据
- 同步游戏分数和成就解锁状态
