## 页面功能详细说明

### 1. index.html - 主页面（单词学习模式）
**核心功能**:
- 导航栏：网站Logo、主要页面链接
- 简洁的标题区域：网站名称和简介
- 单元选择区域：8个单元的卡片式选择界面
- 单词学习区域：
  - 左侧：单词卡片（显示英文、音标、发音按钮）
  - 右侧：释义面板（显示中文释义、词性、例句）
  - 底部：学习控制（上一个/下一个、难度标记、进度条）
- 学习统计：今日学习单词数、掌握程度分布

**交互组件**:
- 单词卡片翻转动画
- 单元切换滑动效果
- 发音播放功能
- 难度标记系统（简单/中等/困难）

### 2. practice.html - 练习测试页面
**核心功能**:
- 测试类型选择：英译中、中译英、听力测试、完形填空
- 题目设置：题目数量选择（10/20/30题）
- 测试界面：
  - 题目显示区域
  - 选项按钮（4选1）或输入框
  - 计时器
  - 进度指示器
- 结果页面：
  - 得分显示
  - 错题回顾
  - 正确答案解析
  - 重新测试按钮

**交互组件**:
- 题目切换动画
- 答题反馈效果（正确/错误）
- 计时器动画
- 结果统计图表

### 3. games.html - 单词游戏页面
**核心功能**:
- 游戏类型选择：
  - 单词消消乐：匹配相同释义的单词
  - 单词拼图：字母重新排列
  - 记忆翻牌：匹配单词和释义
  - 单词接龙：根据最后一个字母连接
- 游戏设置：难度级别、单元范围
- 游戏界面：
  - 游戏区域
  - 计分板
  - 计时器
  - 暂停/重新开始按钮
- 游戏结束：得分显示、排行榜、再玩一次

**交互组件**:
- 拖拽交互（消消乐）
- 点击翻转效果（翻牌）
- 粒子动画效果
- 得分动画

### 4. progress.html - 学习进度统计页面
**核心功能**:
- 学习概览：
  - 总学习单词数
  - 已掌握单词数
  - 学习天数
  - 平均每日学习时长
- 可视化图表：
  - 单元掌握度雷达图
  - 每日学习趋势折线图
  - 单词难度分布饼图
  - 测试成绩柱状图
- 成就系统：
  - 学习里程碑徽章
  - 连续学习奖励
  - 特殊成就解锁
- 错误本：易错单词列表和重点复习建议

**交互组件**:
- 图表动画效果
- 数据筛选器（按单元、时间范围）
- 成就解锁动画

## JavaScript模块划分（main.js）

### 数据管理模块
- `WordData`: 单词数据结构和操作
- `ProgressManager`: 学习进度管理
- `StorageManager`: 本地存储操作

### UI组件模块
- `CardComponent`: 单词卡片组件
- `QuizComponent`: 测试题目组件
- `ChartComponent`: 图表可视化组件

### 游戏逻辑模块
- `MatchingGame`: 消消乐游戏逻辑
- `PuzzleGame`: 拼图游戏逻辑
- `MemoryGame`: 记忆翻牌逻辑

### 工具函数模块
- `AnimationUtils`: 动画效果工具
- `AudioUtils`: 音频播放工具
- `ValidationUtils`: 数据验证工具

## 资源文件说明

### 图片资源
- `hero-bg.jpg`: 主页背景，选择教育、学习相关的温馨图片
- `study-icon.png`: 学习模式图标，简洁的书本或铅笔图标
- `game-icon.png`: 游戏模式图标，有趣的游戏手柄或拼图图标

### 数据结构
单词数据将以JSON格式存储，包含：
```javascript
{
  "unit": 1,
  "word": "golden",
  "phonetic": "/ˈɡəʊldən/",
  "translation": "金的；金色的",
  "pos": "adj.",  // 词性
  "example": "The golden crown was beautiful.",
  "difficulty": "medium"  // 难度等级
}
```

## 技术栈集成

### 核心库使用
1. **Anime.js**: 页面切换、卡片翻转、按钮交互动画
2. **ECharts.js**: 学习进度图表、成绩统计可视化
3. **p5.js**: 背景粒子效果、单词云展示
4. **Splide.js**: 单元切换轮播、单词卡片滑动
5. **Matter.js**: 消消乐游戏物理引擎
6. **PIXI.js**: 游戏图形渲染和特效

### 响应式设计
- 使用Tailwind CSS实现响应式布局
- 移动端优化的触摸交互
- 不同屏幕尺寸的自适应调整
